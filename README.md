# Currículo Interativo - Iago de Souza Gomes

## 🚀 Sobre o Projeto

Este é um projeto de currículo interativo desenvolvido para apresentar minhas qualificações, experiências e projetos de uma forma moderna e dinâmica. A navegação é baseada em scroll entre seções, criando uma experiência de usuário fluida e contínua.

**[Clique aqui para ver o site ao vivo!](https://IagoSet.github.io/CurriculoInterativo/)**

## ✨ Funcionalidades

-   **Navegação por Seções:** Navegue facilmente entre as seções "Sobre", "Formação", "Experiência", etc.
-   **Tema Claro/Escuro:** Alterne entre os modos claro e escuro para melhor conforto visual.
-   **Design Responsivo:** O layout se adapta a diferentes tamanhos de tela.
-   **Botão "Voltar ao Topo":** Navegue rapidamente de volta ao início da página.

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

-   **HTML5:** Para a estrutura e semântica do conteúdo.
-   **CSS3:** Para estilização, layout e animações.
-   **JavaScript (ES6+):** Para a interatividade, como a troca de tema e o botão de scroll.
